# AI Agent Skills

This repository will contain reusable skills that extend and customize how AI agents work. Each skill will package focused guidance, workflows, or supporting resources that agents can apply to specific tasks.

## Skills

Document each skill using the following template:

### Skill name

**Purpose:** What the skill helps an agent accomplish.

**When to use it:** The situations or tasks that should trigger the skill.

**How it works:** A brief overview of its instructions, workflow, and included resources.

**Location:** `path/to/skill/`

**Status:** Planned, in development, or ready to use.
